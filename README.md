# 🌡️ Temperature Converter Website

## 🎯 Objective
The objective of this project is to develop an interactive web application that converts temperature values between **Celsius**, **Fahrenheit**, and optionally **Kelvin**.  
This project demonstrates the fundamental concepts of **front-end web development** including layout design, styling, and dynamic interaction using JavaScript.

---

## 🛠️ Technologies Used

- **HTML5** – For structuring the web page
- **CSS3** – For styling and responsive design
- **JavaScript** – For implementing temperature conversion logic and DOM manipulation

---

## 📌 Features

- User-friendly input field for entering temperature
- Dropdown menu to select temperature unit
- Real-time temperature conversion
- Input validation for numeric values
- Clean and modern UI design
- Responsive layout for different screen sizes

---

## 📋 Steps Performed

1. Designed a clean and intuitive user interface using **HTML and CSS**.
2. Created an input field and dropdown menu for selecting temperature units.
3. Implemented the temperature conversion logic using **JavaScript**.
4. Added validation to ensure only numeric values are accepted.
5. Displayed converted temperature dynamically on the screen.
6. Styled the interface using **modern gradients and responsive layout techniques**.

---

## ✅ Project Outcome

The final result is a **functional and user-friendly temperature converter** that demonstrates key JavaScript concepts such as:

- Event handling
- DOM manipulation
- Input validation
- Dynamic UI updates
